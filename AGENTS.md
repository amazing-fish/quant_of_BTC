# 项目协作规范

- 语言为中文，包括提交信息、PR 描述以及文档。
- Python 代码遵循 [PEP 8](https://peps.python.org/pep-0008/) 与 [PEP 257](https://peps.python.org/pep-0257/) 文档字符串规范，必要时可使用类型注解。
- 优先编写纯函数与小体量模块，避免重复逻辑；若必须重复，请抽取辅助函数。
- 回测逻辑需保持可重复性，任何新增依赖请更新 README。
- 所有新增脚本需提供最少一段使用说明或示例。
- 提交前需要运行并明确给出当前策略的运行结果，持续覆盖README中的结果部分
- 根据项目进展更新README，必要时更新AGENTS
